# Documentation

This directory contains design notes, supporting figures, and synthesis reports.

## Contents

- `docs/`: implementation notes and precision-roadmap documentation.
- `tex_note/`: non-manuscript LaTeX notes and support images.
- `synthesis_reports/`: HLS synthesis reports copied from local synthesis
  runs.

The manuscript drafting directory `paper_writing/` is intentionally excluded
from Git and is not part of this documentation tree.
