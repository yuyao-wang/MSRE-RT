# Synthesis Reports

The `windows_hls_reports/` folder contains the available HLS report artifacts
from the local Windows/Vivado HLS runs:

- core step kernel reports and schedule XML/SVG artifacts.
- BOP step kernel reports and schedule XML/SVG artifacts.
- N200/S1 and N80/S1 report variants where available.

These files are documentation artifacts only. Build directories and regenerated
HLS work directories remain ignored by Git.
